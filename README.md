# CRM Dashboard UI/UX Design

## 📌 Project Overview

This project is a web-based CRM (Client Relationship Management) dashboard designed for a digital agency to manage leads, clients, and tasks efficiently.

---

## 🏢 Real-World Scenario

Designed for:
**A digital agency managing multiple clients and leads**

Problems:

* Managing leads using spreadsheets
* Missing follow-ups
* Scattered communication (WhatsApp, emails)

Solution:
A centralized CRM dashboard to streamline workflow and improve efficiency.

---

## 🎯 Key Features

### Dashboard

* Overview of leads, revenue, tasks, and clients
* Helps in quick decision-making

### Lead Pipeline

* Stages: New → Contacted → Qualified → Proposal → Converted
* Tracks lead progress clearly

### Clients Page

* Displays client list with status and budget
* Includes actions to view details

### Client Profile

* Shows detailed client information
* Helps track engagement and communication

### Tasks & Follow-ups

* Tracks tasks with status (Pending, Done, Overdue)
* Ensures no follow-ups are missed

---

## 🧠 Design Decisions

* Simple and clean layout for daily usage
* Clear status indicators for quick understanding
* Sidebar navigation for easy access
* Focus on real agency workflow
